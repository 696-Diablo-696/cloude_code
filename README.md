# cloude_code

Crea una página web profesional de una empresa llamada SERVITIC de servicios
tecnológicos y mantenimiento informático.
La web debe tener:
• Cabecera de ancho completo con logo (20%) y menú (80%)
• Sección Servicios
• Sección Quiénes somos
• Sección Contacto con formulario con los campos básicos
• Pie de página que incluya el email, teléfono y un menú LOPD con política de
privacidad, cookies y aviso legal
Usa HTML5 semántico y CSS externo bien organizado. Diseño limpio, centrado y
responsive.
